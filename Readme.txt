# GrowEasy AI CSV Importer


## Features

- CSV drag and drop upload
- AI powered CRM extraction
- Gemini/OpenAI integration
- Dynamic column mapping
- Batch processing
- Dark mode


## Tech Stack

Frontend:
Next.js

Backend:
Node.js
Express

AI:
Gemini


## Setup


Backend:

cd backend

npm install

create .env

GEMINI_API_KEY=your_key

npm run dev



Frontend:

cd frontend

npm install

npm run dev
